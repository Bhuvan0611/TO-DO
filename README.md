# 📝 To-Do List App

A simple and interactive **To-Do List** built using **HTML, CSS, and JavaScript**.  
Easily add, delete, and manage your daily tasks with a clean interface.

---

## 🚀 Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 💾 Auto-saves tasks using Local Storage
